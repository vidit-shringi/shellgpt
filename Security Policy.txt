Reporting a Vulnerability

As this tool handles API keys and terminal configurations, security is paramount.

If you discover a security vulnerability within this repository (e.g., unintended credential logging, arbitrary code execution flaws in the bash script), please do NOT disclose it publicly. Instead, open a private issue or email the maintainer directly. We will review the vulnerability and issue a patch as quickly as possible.
